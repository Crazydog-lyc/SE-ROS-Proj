# 测试计划

**项目名称**：SE-ROS-Proj  
**日期**：2026-06-18  
**版本**：2.0

---

## 1. 测试目标

验证四类自研模块（FR-A~D）及集成 launch 满足需求分析报告中的验收标准，并保证 batch 流水线可重复执行。

---

## 2. 测试范围

| 层级 | 范围 | 工具 |
|------|------|------|
| 单元测试 | mission 状态机/loader/adapter；scenario metrics；semantic 几何/代价 | pytest, gtest |
| 集成测试 | safety 服务；mission+safety 状态联动 | launch_test |
| 系统测试 | mission_demo；单 case launch；GUI 导航 | ros2 launch |
| 回归测试 | batch 流水线 | run_batch.sh |

**不在范围**：Nav2 上游功能、Gazebo 物理引擎本身。

---

## 3. 测试环境

| 项 | 要求 |
|----|------|
| OS | Ubuntu 22.04 |
| ROS | Humble (`/opt/ros/humble/setup.bash`) |
| 构建 | `colcon build --symlink-install` |
| 显示 | GUI 测试需 `DISPLAY`；headless 用 `RUN_HEADLESS=True` |
| 超时 | 单 case 默认 900s；Nav2 ready 420s |

## 6. 系统 / 批量测试计划

### 6.1 手动 Demo（ST-01）

```bash
python3 src/sam_bot_nav2_gz/scripts/generate_scattered_room.py --seed 42 --box-count 12
ros2 launch course_bringup mission_demo.launch.py \
  world_file:=demo_pillar_room.sdf enable_semantic:=True enable_safety:=True
```

**通过准则**：Gazebo/RViz 启动；Nav2 active；可完成导航。

### 6.2 单 Case（ST-02）

```bash
ros2 launch nav2_scenario_runner run_single_case.launch.py \
  scenario_file:=/tmp/nav2_cases/pillar_round_001_scenario.yaml \
  run_headless:=False enable_safety:=True
```

**通过准则**：`auto_run_mission` 退出码 0；log 含 `Mission succeeded`。

### 6.3 全量 Batch（ST-03）

```bash
ENABLE_SEMANTIC=True ENABLE_SAFETY=True RUN_HEADLESS=False \
bash src/nav2_scenario_runner/scripts/run_batch.sh
```

**通过准则**：`results.csv` 全部 PASS；每 case 有独立 log。

### 6.4 Batch Profile

| Profile | 文件 | Case 数 | 说明 |
|---------|------|---------|------|
| default | `default_batch.yaml` | 2 | pillar 房间两轮 corridor |
| full | `full_batch.yaml` | 4 | corridor / room / congestion / fault |

## 7. 测试数据

- **输入**：`batch_profiles/*.yaml`、生成的 `/tmp/nav2_cases/*`
- **输出**：`/tmp/nav2_results/{case_id}.log`、`{case_id}.json`、`results.csv`、`summary.json`
- **日志解析关键字**：`Ready for navigation!`、`Mission succeeded:`、`Mission failed:`

---

## 8. 通过/失败准则

| 类型 | 通过 |
|------|------|
| 单元 | 全部 assert 通过；语句覆盖率 ≥ 90% |
| 集成 | 预期状态/话题行为一致 |
| 系统 | mission 成功完成导航任务 |
| Batch | `success=true` 且 `mission_succeeded=true` |
